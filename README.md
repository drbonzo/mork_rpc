# Mörk RPC RPC

What is **Mörk**? 

Mork is a RPC protocol + PHP library.
Data is transfered as JSON objects.


# Features

* PHP RPC library
* JSON data transfer
* Improvement to JSON RPC
* Single requests
* Improved error handling
